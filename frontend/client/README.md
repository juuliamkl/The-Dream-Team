# The frontend client application

This directory contains all the code used to generate the frontend client application. The application is built with React and Vite. To locally serve the application use:
```shell
npm run dev
```
To build this to a static application use:
```shell
npm build
```
To run all the tests and get the coverage use:
```shell
npm run coverage
```

# Contents

This directory contains all the different UI components.

## Header
Defines the header component for the application used by the base page.

## Label
Defines a generic label component. Used for displaying all the different applied and selected projects for each student.

## Project Select
Defines an element that can be clicked to set the current project and then to navigate to sorting said project.

## Score
Contains the definitions of two components related to each other. Firstly, this contains the score component used to display the applicants score, but also the score info component used to give information about the score component to the user.

## Sort Column
Defines the column component between which we can drag applicants.

## Sort Dropdown
Defines the dropdown that sets the current sorting method for all the applicants.

## Student Card
Defines the component used to display the applicant's name, score and labels.

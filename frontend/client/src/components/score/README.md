# Contents
This folder contains the code that defines the score and score info component.

## Score component
The files used for defining the score component.

### score.component.scss
Styling for the component.

### score.component.tsx
The reactive component.

## Score info component
The files used for defining the score info component.

### score-info.component.scss
Styling for the component.

### score-info.component.tsx
The reactive component.

### info-icon.svg
The icon used for the button used to toggle the score info.

### score.svg
Graphic used to explain the score component.

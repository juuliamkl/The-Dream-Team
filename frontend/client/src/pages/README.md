# Contents

This directory contains all the different pages used in the application.

## Base

This is simply the base page for the application. Each different route will be rendered within this page.

## Error

A simple error page.

## Select

The main page of the application. Mainly provides the view to pick a project to build a team for.

## Sort

The page where all the team building happens and where applicants are sorted to to projects.

## Training

The page used for training and initializing the machine learning model as well as setting the global team size.

# Contents

This folder contains all the common types used in the whole application.

## Auth.ts

Types for authetication with the backend.

## Columns.ts

Types used in creating the the different columns on the Sort page.

## Dragging.ts

Types for handling the dragging and drag events on the Sort page.

## Labels.ts

Types utilized in setting the labels in persistant storage.

## ML.ts

Contains all the types related to handling the machine learning model, it's training, etc.

## Project.ts

Defines the types related to getting the project data from the backend.

## Score.ts

All the types related to getting the scores and team from the backend.

## SortMethods.ts

Contains all the different sorting methods and defines the type of a sorting function used in sorting the applicants on the Sort page.

## Student.ts

All the types related to the student.
